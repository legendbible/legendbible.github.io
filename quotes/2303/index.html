もし私たちが違う言語を話していたら、私たちはある程度異なる世界を知覚するだろう。

If we spoke a different language, we would perceive a somewhat different world.
ルートヴィヒ・ウィトゲンシュタイン「哲学的探究」

言語哲学の巨人ウィトゲンシュタインが言語と世界観の関係を語った言葉だ。私たちは言語で世界を切り分け、名付けることで認識する——だから使う言語が違えば、見えてくる世界が変わる。新しい言語を学ぶことは、新しい世界を開くことだ。あなたは今日、新しい言葉に出会い、新しい世界の扉が開いたか。

ウィトゲンシュタイン
Ludwig Wittgenstein ／ 1889 - 1951 ／ 哲学者
オーストリア生まれの哲学者。「論理哲学論考」「哲学的探究」で20世紀言語哲学に革命をもたらした。「語り得ぬものについては沈黙しなければならない」という言葉で知られる。建築・音楽・工学にも才能を持ち財産を全て寄付して小学校教師になるなど型破りな人生を送った。
