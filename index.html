収録名言

レジェンドたちの言葉を、毎日のエネルギーに

ブッダ、孔子、レオナルド・ダ・ヴィンチ、スティーブ・ジョブズ、アインシュタイン、坂本龍馬、マザー・テレサ——。古今東西・時代を超えて人々の心を動かしてきた偉人たちの名言を、いつでもどこでも手軽に楽しめるアプリです。

毎日触れることで、前向きな気持ちになれる。落ち込んだ時に背中を押してくれる。そんな「心のサプリメント」としてLEGEND BIBLEは生まれました。

読みやすく、使いやすいUI/UX

名言アプリはたくさんありますが、「読みにくい」「操作が複雑」という声をよく耳にします。LEGEND BIBLEは徹底的にシンプルで直感的な操作性にこだわりました。

タップひとつで名言の詳細を見られ、スワイプで次の名言へ。カテゴリ別・偉人別の検索も簡単です。毎日継続して使いたくなるデザインを目指しています。

502人のレジェンド、2,084件の名言

古代ギリシャの哲学者から幕末の志士、近代の科学者、現代のアスリート・起業家・アーティストまで、502人の偉人による2,084件の名言を収録しています。東洋・西洋・中東・アフリカ・ラテンアメリカと世界各地の偉人を幅広く網羅しています。

やる気・仕事・心・人間関係・自己成長・時間・挑戦・逆境の8カテゴリで整理されており、その日の気分や状況に合わせた名言を探すことができます。今後も継続的に偉人と名言を追加していきます。

あなたに合った診断機能

「名言診断」は3つの質問に答えるだけで、今のあなたの気持ちに刺さる名言を見つけます。「偉人診断」は5つの質問から、あなたと最も価値観が共鳴する偉人を502人以上の中から導き出します。同じ回答でも毎回異なる偉人が現れる、ランダム性のある診断をお楽しみください。

偉人検索では、分野（哲学者・科学者・芸術家など）・時代・地域でタグ絞り込みができ、50音順や時代順での並び替えにも対応しています。

毎日の習慣に

朝の通勤中に今日の名言を読む。仕事に行き詰まったときにランダム名言でリフレッシュする。就寝前にお気に入りの名言を読み返す。LEGEND BIBLEは、毎日の生活に小さな豊かさを届けることを目指しています。

お気に入り登録・閲覧履歴・いいね機能で、あなただけの名言コレクションが作れます。「今日の名言」は日替わりでおすすめの5件をお届けします。

名言の力を信じて

「千里の道も一歩から」（老子）、「We can only see a short distance ahead, but we can see plenty there that needs to be done.」（チューリング）——偉人たちの言葉には、時代を超えた普遍的な知恵が宿っています。一日一言、レジェンドたちの言葉をあなたの日常に取り入れてみてください。

はじめに

LEGEND BIBLE（以下「本アプリ」）は、ユーザーのプライバシーを尊重し、個人情報の保護に努めます。本ポリシーは、本アプリにおける情報の収集・利用・管理について説明します。

収集する情報

本アプリは、以下の情報をお客様の端末内にのみ保存します。

  • お気に入りに登録した名言
  • 閲覧履歴
  • いいね情報
  • 表示設定（フォントサイズ・テーマ）

これらの情報はすべてお客様の端末内（ローカルストレージ）にのみ保存され、外部サーバーへ送信されることはありません。

広告について

本アプリはGoogle AdSenseによる広告を掲載する場合があります。広告配信のためにGoogleがCookieを使用することがあります。Googleによる広告のCookieの使用については、Googleプライバシーポリシーをご参照ください。

アクセス解析

本アプリは、サービス改善のためにアクセス解析ツールを使用する場合があります。取得されるデータは匿名であり、個人を特定するものではありません。

個人情報の第三者提供

本アプリは、法令に基づく場合を除き、収集した情報を第三者に提供することはありません。

プライバシーポリシーの変更

本ポリシーは、必要に応じて変更される場合があります。変更後のポリシーは本アプリ内に掲載した時点で効力を生じるものとします。
